## 📖 프로그램 상세 페이지 (program-detail.html)

### 프로그램 헤더
- 실시간 통계 (수강생, 소요 시간, 레벨)
- 강사 정보 및 평점
- 할인 정보 표시

### 수강 신청 카드
- 가격 정보 (할인율 표시)
- 시작일, 시간, 방식 안내
- 수강 신청 / 무료 체험 버튼
- 이미 수강한 학생 수 표시

### 5개 탭 구조
1. **개요**: 프로그램 소개, 추천 대상, 학습 결과물
2. **커리큘럼**: 8주 상세 일정 (Week별 내용)
3. **강사 소개**: 약력, 전시, 전문 분야
4. **수강 후기**: 평점 4.9, 학생 리뷰
5. **FAQ**: 자주 묻는 질문 (접기/펼치기)

### 모달 기능
- 수강 신청 모달 (이름, 이메일, 전화번호, 목표 학교)
- 무료 체험 모달
- 폼 검증 및 제출

## 🧮 계산기 페이지 (calculators.html)

### 1. 학비 계산기
**입력 항목**
- 국가 선택 (미국, 영국, 프랑스, 독일)
- 학위 과정 (학사 4년 / 석사 2년)
- 학교 유형 (저렴한 학교 / 평균 / 명문 사립)
- 환율 설정

**결과 표시**
- 연간 학비 (KRW & USD)
- 총 학비 (전체 과정)
- 세부 내역 (수업료, 재료비, 기타)

### 2. 생활비 계산기
**입력 항목**
- 도시 선택 (뉴욕, 런던, 파리, 베를린)
- 거주 형태 (기숙사, 원룸, 쉐어하우스)
- 생활 수준 (검소 ← → 여유)

**결과 표시**
- 월 생활비 + 연간 총액
- Chart.js 도넛 차트 (숙박, 식비, 교통, 기타)
- 항목별 금액 표시

### 3. 장학금 예측기
**입력 항목**
- GPA (0-4.0)
- 포트폴리오 수준 (우수/양호/보통)
- 경력/수상 (다수/보통/적음/없음)
- 재정 상황 (필요/보통/여유)

**결과 표시**
- 예상 장학금 금액 (연간)
- 장학금 확률 (퍼센트 + 프로그레스 바)
- 유형별 가능성 (성적 우수, 재정 지원, 다양성)
- AI 추천사항

### 4. 대출 계산기
**입력 항목**
- 대출 금액 (만원)
- 대출 기간 (1-20년, 슬라이더)
- 연 이자율 (%)
- 상환 방식 (원리금 균등 / 원금 균등)

**결과 표시**
- 월 상환액
- 총 상환액 & 총 이자
- Chart.js 막대 차트 (연도별 상환)
- 초기 3회차 상세 내역
