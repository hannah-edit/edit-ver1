# Edit - 미술 유학 플랫폼

세계적인 아티스트로 성장하는 가장 스마트한 방법

## 🎨 프로젝트 소개

Edit은 AI 기반 맞춤형 컨설팅과 글로벌 아티스트 네트워크를 통해 미술 유학의 꿈을 현실로 만드는 혁신적인 플랫폼입니다.

### 주요 기능

#### 메인 랜딩페이지
- **AI 아트 어드바이저**: 24/7 실시간 상담, 포트폴리오 분석, 합격 예측
- **아티스트 프로그램**: 세계적인 아티스트와의 온라인 강의 및 멘토링
- **성공 사례**: 졸업생들의 생생한 후기와 대화 기회
- **계산기 도구**: 학비, 생활비, 장학금, 대출 계산기

#### 어드민 대시보드
- **대시보드**: 전체 통계 및 실시간 현황
- **아티스트 관리**: 등록, 수정, 삭제 및 프로필 관리
- **Zoom 세션 관리**: 실시간 화상 세션 예약 및 관리
- **학생 관리**: 등록 학생 정보 및 진행 상황 추적
- **컨텐츠 관리**: 교육 자료 업로드 및 관리
- **분석 리포트**: 데이터 분석 및 통계

## 🚀 시작하기

### 파일 구조

```
cursor/
├── index.html              # 메인 랜딩페이지
├── program-detail.html     # 프로그램 상세 페이지
├── calculators.html        # 유학 비용 계산기
├── alumni-stories.html     # 졸업생 스토리
├── admin.html              # 어드민 대시보드
├── script.js               # 메인 페이지 JavaScript
├── admin.js                # 어드민 페이지 JavaScript
├── api-mock.js             # Mock API 데이터
└── README.md               # 프로젝트 문서
```

### 실행 방법

1. **로컬 서버로 실행** (권장)
   ```bash
   # Python 3를 사용하는 경우
   python3 -m http.server 8000
   
   # Python 2를 사용하는 경우
   python -m SimpleHTTPServer 8000
   
   # Node.js를 사용하는 경우
   npx serve
   ```

2. **브라우저에서 직접 열기**
   - `index.html` 파일을 더블클릭하여 브라우저에서 실행
   - 일부 기능은 로컬 서버 환경에서만 정상 작동할 수 있습니다

3. **접속**
   - 메인 페이지: `http://localhost:8000/index.html`
   - 어드민 페이지: `http://localhost:8000/admin.html`
